Добавляет астрономические события в календарь устройства. Является дополнением к приложению Suntimes.

Позволяет добавлять астрономические события в виде отдельных (пользовательских) календарей, которые будут доступны в системе наряду с основным календарем устройства.
Поддерживает солнцестояния, равноденствия, периоды сумерек (астрономических, навигационных, гражданских), восход и заход солнца, положения луны (восход, заход, основные фазы, апогей, перигей), полярные и белые ночи, межсезонные праздники (напр. Самайн, Бельтайн) и другое.

Для корректной работы требуется приложение <a href="https://f-droid.org/en/packages/com.forrestguice.suntimeswidget/">Suntimes версии 0.10.3</a> или выше.

Приложению требуются разрешения:
&bull; <small><u>WRITE_SYNC_SETTINGS</u></small> для добавления пользовательских календарей в системное приложение «Календарь»;
&bull; <small><u>WRITE_CALENDAR</u></small> для добавления событий в пользовательские календари;
&bull; <small><u>SUNTIMES.PERMISSION.READ_CALCULATOR</u></small> для чтения данных из основного приложения Suntimes.
